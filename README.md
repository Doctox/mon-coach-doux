# Mon Coach Doux

Prototype d'application d'accompagnement pour une routine quotidienne douce de callisthenie.

L'app propose :

- choix du mode Brise, Doux ou Sombre ;
- chrono exercice par exercice ;
- consignes detaillees ;
- voix de guidage si le navigateur la supporte ;
- pause, passage d'exercice et adaptation en cas de douleur ;
- installation possible sur Android via Chrome et GitHub Pages.

## Installation sur Android

Une fois le site publie avec GitHub Pages :

1. Ouvrir le lien dans Chrome sur Android.
2. Ouvrir le menu avec les trois points.
3. Choisir "Ajouter a l'ecran d'accueil" ou "Installer l'application".
